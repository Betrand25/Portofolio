Lego Store is an Android-based application designed to make it easier for users to view various types of the latest Lego available. Through this application, users can not only browse the latest Lego collections easily, but also have the opportunity to buy Lego online without having to go to a physical store. Apart from that, this application also provides a feature to view Lego store locations online, so users can quickly find the nearest store if they want to shop directly or see a more complete collection. With complete and easy-to-use features, this application aims to provide a practical and enjoyable Lego shopping experience for Lego fans.

What was your role in the group?

This task was completed collaboratively in a group, with my specific responsibilities including designing and developing the home bucket and payment features, creating the database, and integrating the Google Maps API into the application.

What are the benefits of the project

This application provides various significant benefits for users. With the online shopping feature, users can easily explore the latest Lego collections and make purchases without having to visit a physical store, saving time and energy. In addition, Google Maps API integration allows users to practically find the location of the nearest Lego store, making it easier for those who want to shop directly or view a more complete collection. This application is designed to provide a convenient, fast and efficient shopping experience, especially for Lego fans.
